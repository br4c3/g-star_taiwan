# 개항장 B2B 로컬기프트 선행연구 검토

## 1. 검토결론

선행연구는 개항장 지역자산의 조사·기록·스토리텔링 필요를 지지하지만 B2B 로컬기프트의 구매수요와 수익성을 증명하지 않는다. 따라서 문헌은 지역성·조사방법·윤리 기준에 사용하고, 시장성은 인터뷰·견적·발주로 검증한다.

## 2. 핵심 연구

### 2.1 개항장 문화유산 활용

민경선(2025)은 개항장 근현대문화유산의 소유·운영주체가 달라 통합 활용에 제약이 있다고 진단한다. 본 사업은 시설을 통합하지 않고 상품·생산자·지역서사와 거래를 연결하는 제한된 역할을 맡는다.

- [인천연구원 공식 소개](https://www.ii.re.kr/base/board/read?boardManagementNo=14&boardNo=18130)

### 2.2 개항장 장소자산 스토리텔링

김창수의 연구는 개항장 활성화계획에 비해 장소성 회복을 위한 체계적 스토리텔링 조사가 부족하다고 본다. 상품카드에 출처, 확인일, 상인검수와 불확실성을 표시하는 근거다.

- [인천연구원 공식 소개](https://www.ii.re.kr/base/board/read?boardManagementNo=14&boardNo=15237)

### 2.3 자산지도화

최현선·오윤경·이지연(2012)은 주민 참여형 PPGIS와 워크숍을 활용한 자산지도화 사례를 제시한다. 상점 후보지도에는 좌표뿐 아니라 상품, 사람, 의미, 활용상태와 검증여부를 포함한다.

- [KCI](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART001724402)
- [보관 PDF](sources/papers/krihs_2012_asset_based_community_activation.pdf)

### 2.4 참여형 자산매핑

Dhital 외(2023)는 같은 자산도 맥락에 따라 긍정·부정 효과를 가질 수 있음을 보여준다. 지역상품을 무조건 좋은 자산으로 분류하지 않고 공급부담, 권리, 편중과 접근장벽을 조사한다.

- [UCL 원문](https://discovery.ucl.ac.uk/id/eprint/10172982/)
- [보관 PDF](sources/papers/dhital_2023_participatory_asset_mapping.pdf)

### 2.5 지역 아카이빙

권순명·이승휘(2009)는 공공기록만으로 지역생활을 충분히 재현하기 어렵다고 본다. 상인 인터뷰·상품·사진을 기록하되 동의·검수·철회범위를 관리한다.

- [KCI](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART001365321)
- [보관 PDF](sources/papers/ksas_2009_regional_archiving.pdf)

### 2.6 항만도시 기록화

권도균·장우권(2024)은 지리, 비물질 정보, 산업경제, 변화, 참여와 행정 등 항만도시 기록요소를 제시한다. 개항장 상품의 설명을 건축 외관에 한정하지 않고 교역·노동·생산자·변화를 함께 기록한다.

- [KCI](https://www.kci.go.kr/kciportal/landing/article.kci?arti_id=ART003053418)
- [보관 PDF](sources/papers/kslis_2024_port_city_regeneration_documentation.pdf)

### 2.7 관광기념품 연구의 전체 구조

Shen·Lai(2022)는 1981~2020년 관광기념품 연구 248편을 검토해 상품 자체, 경제·사회문화 효과, 사업생태계, 구매행동, 구매만족의 다섯 연구축을 제시한다. 이는 본 사업이 지역서사만 조사해서는 부족하며 상품성, 공급자, 구매담당자와 수령자 경험을 함께 검증해야 한다는 근거다.

- [출판사 원문](https://doi.org/10.1177/21582440221106734)

### 2.8 문화요소와 구매의도

Huang 외(2025)는 7개 시나리오 실험, 총 3,203명의 응답을 통해 관광기념품에 담긴 전통문화 요소와 인지가치·문화정체성·구매목적의 관계를 분석했다. 연구결과는 출처가 확인된 개항장 문화요소를 패키지에 반영할 가설을 지지하지만, 가상상황 기반 중국 표본이므로 인천 B2B 구매의도로 일반화하지 않는다.

- [PLOS ONE 원문](https://doi.org/10.1371/journal.pone.0313905)
- [보관 PDF](sources/papers/huang_et_al_2025_cultural_load_souvenir.pdf)

### 2.9 전통 재현보다 현대적 사용성

Xie·Wu·Hsieh(2012)의 대만 연구는 전통형태를 그대로 복제한 상품보다 현대적 디자인과 원주민 문화표지를 결합한 상품이 더 진정성 있게 인식될 수 있음을 보고했다. 강민지·전성복(2007)은 국내 촬영지 기념품 조사에서 색상, 실용성, 품질, 휴대·보관 편의의 중요성을 제시했다. 따라서 개항장 키트는 옛 이미지를 많이 넣는 경쟁이 아니라 현대적 사용성, 품질과 지역표지의 적합성을 함께 시험해야 한다.

- [Xie 외 논문](https://doi.org/10.1080/10548408.2012.691400)
- [강민지·전성복 KCI](https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART001201729)

### 2.10 획일적 지역상품의 위험

김세웅(2023)은 타지역 사례에서 어디서나 구매할 수 있는 전통요소와 먹거리 중심의 획일화를 문제로 지적했다. 인천 개항장 적용 시에는 항구·근대건축 이미지만 반복하지 않고 실제 상점, 생산자, 교역·이주·생활사를 상품별로 구분해 제시해야 한다.

- [KCI 원문](https://www.kci.go.kr/kciportal/ci/sereArticleSearch/ciSereArtiView.kci?sereArticleSearchBean.artiId=ART002942460)

## 3. 사업에 반영할 항목

| 연구영역 | 반영 |
|---|---|
| 자산매핑 | 위치·상품·사람·근거·권리·공급능력 |
| 스토리텔링 | 주장별 출처·상인검수·불확실성 |
| 지역아카이빙 | 인터뷰 원본·공개범위·철회절차 |
| 항만도시 기록 | 교역·이주·노동·산업변화 포함 |
| 참여방법 | 상점을 콘텐츠 대상이 아닌 공동검수자로 참여 |
| 제품설계 | 지역표지·현대적 사용성·품질·휴대성 동시 평가 |
| 구매검증 | 문화적 매력과 B2B 발주조건을 분리해 검증 |

## 4. 문헌이 답하지 못하는 질문

- 구매기관의 선물예산과 평균수량
- 통합납품 문제의 심각도
- 표준키트의 지불의사
- 세트당 실제 공헌이익
- B2B 재주문율

이 다섯 항목은 현장실증에서만 답한다.
