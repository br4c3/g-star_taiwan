# 대회 목적과 개항장 B2B 로컬기프트 연구방향

## 1. 대회의 목적

이 과제는 앱 아이디어를 제안하는 것이 아니라 지역자산을 사회문제 해결, 사용자 경험, 지속 가능한 거래와 구현 가능한 운영체계로 연결하는 데 목적이 있다.

> 지역자산 발굴 → 사회문제 정의 → 4대 가설 → 사용자 경험 → 비즈니스 모델 → 기술·실행 타당성 → 사회성과

## 2. 본 사업의 적합성

| 요구 | 반영 |
|---|---|
| 지역성 | 개항장 역사문화와 현재 상점·상품을 사용 |
| 사회문제 | 작은 상점의 B2B 판로·표준납품 역량 부족 가설 |
| 사용자 | 행사·숙박·기업 구매담당자와 지역상점 |
| 사업성 | 표준키트 판매마진과 맞춤제작비 |
| 실현성 | 5개 상점·1개 상품·20개 고객·12주 실증 |
| 임팩트 | 정상가격 매입액·반복거래·상점별 매출분포 |

## 3. 대회에서 답할 질문

1. 왜 개항장이어야 하는가?
2. 구매기관이 실제로 겪는 조달문제는 무엇인가?
3. 상점은 반복공급할 수 있는가?
4. 누가 얼마를 지불하는가?
5. 30세트 납품 후에도 이익이 남는가?
6. 지역상점에 어떤 측정 가능한 가치가 돌아가는가?
7. 기존 관광앱·기념품·판촉물 업체와 무엇이 다른가?

## 4. 검증원칙

- MICE 행사 수를 로컬기프트 시장규모로 환산하지 않는다.
- “인천다운 선물이 부족하다”는 구매담당 인터뷰 전에는 가설이다.
- 호감·칭찬보다 행사일·수량·예산이 있는 견적요청을 사용한다.
- 발주서·예약금·납품·정산을 최종 증거로 사용한다.
- 목표 미달과 부정적 결과도 연구결과로 기록한다.

## 5. 최종 제출물

- 개항장 지역자산·공급상점 후보 지도
- 구매담당자와 상점 인터뷰 결과
- 표준 웰컴키트 샘플과 상품사양서
- 30·50·100세트 원가·견적표
- B2B 고객여정과 서비스 블루프린트
- 사회적 BMC와 단위경제성
- 12주 WBS와 승인 게이트
- 주문·검수·납품·정산 시스템 UML
- 유료발주 또는 기각 결과와 사회성과

## 6. 최종 방향

> 개항장 소상공인의 기존 상품과 검증된 지역 이야기를 표준형 웰컴키트로 공동기획하고, 행사·숙박·기업 구매자에게 통합 납품하여 지역의 정상가격 매출과 반복거래를 만드는 B2B 로컬기프트 스타트업을 검증한다.
