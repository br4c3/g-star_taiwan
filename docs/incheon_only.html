REGION OPTION 01

인천 특산물·생산문화 발견 연구

바다·섬·이주·산업문화에서 만들어진 상품과 생산자를 발견 가능한 콘텐츠로 전환한다.

소셜미션: 인천의 지역성을 담은 작은 생산자와 상품을 새로운 소비자에게 연결하여 생산문화가 지속되도록 한다.

1. 문제정의

인천에는 포구 수산물, 섬 생산물, 개항기 다문화 식문화와 산업기술 기반 공예가 존재한다. 그러나 상품·생산자·장소·구매처 정보가 흩어져 소비자는 지역성을 판단하기 어렵고 생산자는 자신의 이야기를 전달할 콘텐츠 역량이 부족하다.

2. 초기 발굴 주제

바다와 섬의 생산물

후보소래포구 수산가공품, 강화·옹진 섬 농수산물, 지역 식문화
검증원산지·가공장소·생산자·반복공급과 기존 판매처

개항과 이주의 식문화

후보개항장·차이나타운 주변의 오래된 제조·조리 기술과 지역 브랜드
검증관광상품과 다른 지역성, 생산자 동의, 체험·구매 가능성

산업기술 기반 로컬제품

후보지역 제조기술·부산물·공방을 활용한 공예와 생활제품
검증지역 내 생산 비중, 품질, 소량생산과 가격 경쟁력

3. 스타트업 실행범위

  • 후보 자산 10~20개와 생산자 3~5곳 조사
  • 생산자 이야기·장소·기존 구매처를 연결한 모바일 발견지도
  • 주제형 체험 또는 소량 발견키트 1개 선주문 실험
  • 자체 쇼핑몰·대량재고·기업복지 계약은 제외

4. 검증지표

가설지표
발견비보조인지, 신규 자산 발견률
이해지역성 핵심사실 회상
행동상세조회, 구매처 클릭, 체험예약
지불예약금·선주문 건수와 가격수용성
공급생산자의 반복공급·공동제작 의사
